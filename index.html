Timestamp Microservice!

I can pass a date or unix code as a parameter in the url and it will check to see whether that string contains either a unix timestamp or a natural language date (ex. August 8, 1994)

Example usage:

https://amp-timestamp.glitch.me/December%2015,%202015
https://amp-timestamp.glitch.me/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
